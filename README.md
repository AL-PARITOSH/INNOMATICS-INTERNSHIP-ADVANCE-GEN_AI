# INNOMATICS – ADVANCE GEN AI INTERNSHIP  
## Python Logic Building & System Design Repository  

This repository contains structured implementations developed during the Advance Gen AI Internship at Innomatics Research Labs.  

The project demonstrates progressive learning — starting from core logic building, moving to modular function-based programming, and finally implementing object-oriented system design.

The objective of this repository is to strengthen problem-solving ability, improve clean coding practices, and simulate real-world programming scenarios.

---

# 📌 Project Structure Overview

The repository is organized conceptually into three development levels:

| Level | Coverage | Concepts Used |
|--------|----------|--------------|
| Problems 1 – 3 | Core Logic Building | Input handling, loops, conditionals |
| Problems 4 – 6 | Functions & UDF | Modular programming, reusable logic |
| Problem 7 | OOP Implementation | Classes, objects, encapsulation |
